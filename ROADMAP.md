# ROADMAP (v1)

Questa roadmap è intenzionalmente breve: serve come “bussola” per sprint piccoli e production-safe.

## Milestone 1 — Operability & Governance
- Baseline git + governance minima (sprint, handoff, anti-regression).
- Release checklist legata a `deploy/push-ui.sh`.

## Milestone 2 — Smart Display UI polish
- Clock dominance (idle + deep idle) e gerarchia widget.
- Touch feedback premium, kiosk ergonomics, zero overflow 1280×800.

## Milestone 3 — Weather reliability & portability
- Location: GPS primario quando disponibile, override manuale stabile.
- Forecast 7 giorni affidabile e leggibile (UI non vuota su errori rete).

## Milestone 4 — Assistant integration (incrementale)
- Contratto API + health/streaming definiti.
- Conversazione UI con fallback offline.

## Milestone 5 — Medical screen (scope controllato)
- Screen dedicato con dati reali solo dopo definizione privacy/sicurezza.
